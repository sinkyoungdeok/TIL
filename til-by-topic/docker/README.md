## 1. 설치 명령어 

### Docker 설치 명령어 (Mac OS)

```
brew install --cask docker
```

### Kubectl 설치 명령어 (Mac OS)

```
brew install kubectl 
```

### Kustomize 설치 명령어 (Mac OS)

```
brew install kustomize
```

### Minikube 설치 명령어 (Mac OS)

```
brew install minikube
```
